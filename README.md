shymean
===

个人博客[www.shymean.com](http://www.shymean.com)
## 相关指令
开发
```
# nodemon监听node服务端
npm run dev:server

# webpack启动ssr
npm run dev:ssr

```
打包
```
# webpack打包静态资源
npm run build:ssr
# tsc打包服务端
npm run build:server
```

部署
```
# pm2启动服务端
npm run start
```

上传博客
```
# 上传本地markdown博文
npm run upload /path_to_markdown_file/xxx.md
```
